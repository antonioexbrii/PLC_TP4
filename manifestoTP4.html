Resumo:

A quarta entrega consiste na conversão dos links do Arquivo Musical para os de um servidor que, enquanto a correr, faz a seleção das páginas a ser mostradas em vez da chamada a ficheiros locais. A navegação pelo site é gerida unicamente pelo servidor.

No repositório git estão incluídos os ficheiros de HTML do website e o servidor node.js. Também foram incluídos os ficheiros de conversão XSLT para HTML, XML de origem e manifesto, em ambos os formatos.
